React Nonogram

A nonogram is a logic puzzle consisting of a regular grid, where the objective is to determine which of the grid cells is occupied through the use of clues for each row and column. The clues describe contiguous runs of occupied cells, so that a clue of “3 1 2” indicates a run of 3 contiguous occupied cells, followed by some unknown number of empty spaces, one occupied cell, some number of spaces, and finally two contiguous occupied cells.

You may know nonograms from other products, such as Griddlers, Picross, Picma, or Fugazo World Mosaics.

This nonogram app was written in React by Sean Bolton. The official React Tutorial describes the making of a tic-tac-toe game, but nonograms are way more interesting! You can find the MIT-licensed source code here.

Left-click to mark a occupied cell, right-click to mark a space.
